# Parly docs repo instructions

This repository contains the public Mintlify documentation for Parly.

## Purpose

Use this repo only for public-facing docs content, navigation, and assets.

## Writing rules

- Keep tone clear, professional, and externally legible.

- Write for users, operators, developers, and agent builders.

- Keep submitted, pending, confirmed, and failed distinct.

- Do not add unsupported features to public docs.

- Keep repo links and install commands accurate.
